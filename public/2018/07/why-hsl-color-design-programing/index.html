CSS – 14 Jul 2018

ชีวิตง่ายขึ้นด้วยระบบสีแบบ HSL ใน CSS

Web Designer และ Frontend Developer แทบทุกคน คงหนีไม่พ้นที่จะต้องทำงานเกี่ยวข้องกับเรื่องของ “สี” อย่างแน่นอน ซึ่งใน CSS Color โดยทั่วไปนั้น จะใช้รูปแบบเป็น RGB Color ในรูปของ rgb() หรือ HEX–Notation

ในบทความนี้ จะมาพูดถึงรูปแบบของสีใน CSS Color อีกรูปแบบหนึ่ง นั่นคือ HSL Color ซึ่งมีข้อดูข้อเสีย และช่วยให้ชีวิตเราง่ายขึ้นได้อย่างไรนั้น ผมจะเล่าให้ฟังครับ

RGB Color ไม่ดียังไง
